# Project Notebooks

## ⚠️ Notebook Rendering Issue

Some of the Jupyter notebooks in this repository may show an **"Invalid
Notebook"** error when opened directly on GitHub. This happens due to
the presence of **`metadata.widgets`** in the notebook metadata, which
GitHub's notebook renderer sometimes cannot process.

Because of this, the notebooks might fail to preview properly on GitHub.

## ✅ Recommended Way to Open the Notebooks

Please use **Google Colab** to view and run the notebooks.

1.  Open the Google Drive folder below.
2.  Select the notebook you want.
3.  Open it using **Google Colab**.

📂 **Google Drive Folder (All Notebooks):**\
https://drive.google.com/drive/folders/15ztYLQiKj-L5cpfjJbRTp55h63So5lHE?usp=sharing

## 💡 Notes

-   All notebooks run correctly in **Google Colab**.
-   If you download them locally, they will also work in **Jupyter
    Notebook / JupyterLab**.
-   The GitHub preview issue is only related to the notebook metadata
    rendering.
